🔧 报警处理方法
① HP5 / 00-：焊接开始信号OFF等待
机器人"焊接开始信号"处于ON状态，安全回路启动。暂时取消"焊接开始信号"即可恢复。
② ——/——：控制面板开关设定异常
焊接法切换开关设成了非"1~3、6"的值。将开关设回正确值即可恢复。
E- / 000：非常停止（紧急停止）
紧急停止端子（CN5的7-8）处于开放状态。确认冷却水是否正常流动，或暂时取消流量开关"开放"状态即可恢复。
④ E- / 100：输入过电压
输入电压在230V以上时（输入电压切换开关SW1置于220V端时，过电压为253V以上），异常显示灯点亮，焊机自动停止。用万用表等测定输入电压，确认是否输入了上述条件值的电压。关闭焊接电源本体电源开关，确认电压后重新起动焊接电源，解除异常。
⑤ E- / 101：掉相异常
输入电源掉相时，异常显示灯点亮，焊机自动停止。请确认输入电源的各相是否正常接通。通过重新起动焊接电源，解除异常。
⑥ E- / 200：输出过电流
焊接中，流通1秒钟以上的过电流，或者持续短路的话，异常显示灯点亮，焊机自动停止。请确认焊接电流是否超过了额定输出电流，导电嘴和母材之间是否接触、是否存在焊接电缆等输出之间的短路。通过重新起动焊接电源，解除异常。
⑦ E- / 300：温度上升异常
焊接电源内部温度过高，异常显示灯点亮，焊机自动停止。这时请打开电源开关设置冷却期间，使风扇转动10分钟以上。之后关闭电源开关，重新起动后异常解除。再次开始焊接时，请降低负载持续率、焊接电流，进行焊接。
⑧ E- / 301：逆变器驱动回路异常
逆变器驱动回路（印刷板P6768S）的周边存在异常的时候，异常显示灯点亮，焊机自动停止。从新起动焊接电源，解除异常。
⑨ E- / 302：1次·2次电流检测异常
电流检测部存在异常的时候，异常显示灯点亮，焊机自动停止。这时请关闭电源开关，确认印刷板PCB1（P10497P）的端子CN11或者CN14是否脱落。从新起动焊接电源，解除异常。
⑩ E- / 303：±15V输入电压不足异常
控制电源（±15V）的异常、或者输入电压不足。±15V的控制电源中存在异常，或者输入电压在160V以下时（P10497Q上的输入电压切换开关SW1置于220V端时，在175V以下），异常显示灯点亮，焊机自动停止。请关闭焊接电源本体的电源开关，用万用表等测定输入电压，确认是否输入了上述条件值以下的电压。通过再次接通电源开关，解除异常。
⑪ E- / 304：气体检查异常
气体检查开关设定在"检查"侧就输入了焊接开始信号，或者设定在"检查"侧2分钟以上的时候，异常显示灯点亮，焊机自动停止。这时，将气体检查开关设定在"焊接"侧后，就可以恢复动作。
⑫ E- / 305：编码器异常
编码器信号断线和电缆端子接触不良、送丝电机故障等原因造成电机不回转的时候，以及编码器输出的信号中断的时候，异常显示灯点亮，焊机自动停止。从新起动焊接电源，解除异常。
⑬ E- / 307：水压异常
请确认前控制面板的机器人、水冷切换开关是否设定成了"机器人"。将开关设定在"机器人"侧可以恢复动作。使用焊接电源内部的压力开关时，请确认水是否充分流通。取消压力开关的"开放"状态，就能恢复动作。
E- / 400：电弧电压检测线异常
电压检测线上存在异常的时候，启动后异常显示灯点亮，焊机自动停止。请关闭电源开关，确认焊枪侧的电压检测线、以及导电嘴和母材之间加载了不正确的电压时。关闭电源开关后再次接通。
⑮ E- / 401：同期信号线异常
在双丝脉冲MAG焊接法中，同期信号线发生异常的时候，异常显示灯点亮，焊机自动停止。这时请关闭电源开关，确认同期信号线是否断线，焊接电源内部的端子CON6是否连接好、2台电源是否都设定为"双丝脉冲MAG"焊接法。从新起动焊接电源，解除异常。
⑯ E- / 900：微计算机异常（1）
内藏微计算机中发生异常的时候，异常显示灯点亮，焊机自动停止。请关闭电源开关后再次接通。
⑰ E- / 901：波形控制回路异常
波形控制回路（印刷板P10497P）周边存在异常的时候，异常显示灯点亮，焊机自动停止。这时请关闭电源开关后再次接通。
⑱ E- / 920：微计算机异常（2）
内藏微计算机中存在异常的时候，异常显示灯点亮，焊机自动停止。
⚠️ 异常显示时可以操作的功能
发生了异常的时候，前控制面板上显示异常代码。
即使在紧急停止（异常代码：E-000）以及水压异常（异常代码：E-307）的状态下，也可以进行送丝操作。在异常显示状态中进行操作的时候，请务必确认周围的安全后再进行操作。

150T 重点事项：
